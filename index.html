Connection Guide

STEP 01

Android

Download dan install APK Septopia. Tidak perlu aplikasi tambahan.

Get APK
STEP 02

Windows

Buka folder C:/Windows/System32/drivers/etc/hosts dan tambahkan:

91.134.85.13 growtopia1.com
91.134.85.13 growtopia2.com
STEP 03

iOS (Surge)

Install Surge 5 di AppStore dan masukkan URL Proxy di bawah ini:

https://dash.gtps.cloud/ios/5678
STEP 04

macOS

Buka Terminal, ketik sudo nano /etc/hosts lalu paste IP Septopia:

91.134.85.13 growtopia1.com
