🤖 TUTORIAL ANDROID

  1. Pertama, unduh aplikasi Powertunnel dari Playstore/Internet.
  2. Lalu klik tombol "Copy URL Host" pada kolom Android.
  3. Ini akan menyalin tautan host powertunnel otomatis.
  4. Kemudian buka aplikasi PowerTunnel, klik ikon tiga garis, dan klik ikon pengaturan host.
  5. Tempel URL host powertunnel yang telah disalin. Selain itu, ubah periode pembaruan host dari 12 jam menjadi saat dimulai.
  6. Sekarang Anda dapat masuk ke GTPS Anda!

💻 TUTORIAL WINDOWS

  1. Tekan ⊞ Win + R untuk membuka jendela "Jalankan".
  2. Ketik C:\Windows\System32\drivers\etc di kotak teks, lalu Enter.
  3. Temukan file ZIP bernama hosts.
  4. Klik kanan pada file "hosts" dan edit menggunakan Notepad/editor teks.
  5. Tambahkan IP data hosts SPS yang sudah dicopy di baris paling bawah.
  6. Kemudian simpan dan masuk ke Growtopia Anda.

🍎 TUTORIAL IOS

  1. Unduh aplikasi bernama: "Surge 5" dari App Store.
  2. Buka aplikasi lalu Tekan "OK".
  3. Klik pada "Default.conf".
  4. Klik pada bagian "IMPOR" - Unduh Profil dari URL.
  5. Masukkan URL dari hasil salinan tombol host iOS di atas, lalu klik "OK" dan "Selesai".
  6. Tekan "SETUP" lalu setujui kebijakan dengan mengklik "OK" dan "Izinkan" untuk Konfigurasi VPN!.
  7. Selesai, lalu buka "Growtopia" dan Hubungkan!

🍏 TUTORIAL MACOS

  1. Buka aplikasi Terminal di Mac Anda.
  2. Ketik perintah: sudo nano /private/etc/hosts lalu tekan Enter.
  3. Masukkan password administrator Mac Anda apabila diminta.
  4. Gunakan panah keyboard ke baris terbawah, lalu paste data host SPS yang sudah Anda salin.
  5. Tekan Ctrl + O diikuti Enter untuk menyimpan, kemudian Ctrl + X untuk keluar.
  6. Buka game Growtopia Anda dan selamat bermain!
